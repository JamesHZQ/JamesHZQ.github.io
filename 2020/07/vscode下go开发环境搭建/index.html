vscode下GO开发环境搭建

七月 1, 2020 发布在 GO

安装GO

到官网下载相应的包，按照说明安装即可。

  1. 将下载的包解压到/usr/local

    tar -C /usr/local -xzf go$VERSION.$OS-$ARCH.tar.gz

  2. 编辑/etc/profile,添加环境变量

    sudo vim /etc/profile

    添加

    export PATH=$PATH:/usr/local/go/bin

    使其生效

    source /etc/profile

vscode配置

  1. 安装go插件

    新建一个main.go vscode会提示安装相关组件，选择install all。

    注意在此之前一定先做好下面的配置，否则会安装不成功

    go env -w GO111MODULE=on
go env -w GOPROXY=https://goproxy.io,direct

  2. GO的运行与调试

    新版的GO引入了Module，vscode的运行调试方式与老版本略有不同

    1. 首先要在终端初始化当前文件夹，初始化后会生成 .mod文件

      go mod init project-name

    2. 项目运行前，需使用tidy完成 .mod文件依赖包的设置

      go mod tidy

    3. 接下来就可正常使用vscode的调试功能，运行或调试项目了
